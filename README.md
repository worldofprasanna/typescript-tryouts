# Typescript

Playing around with basics of Typescript

To run compile TS to JS in watch mode,

`npx tsc --watch`

To run any file,

`node lib/<file>.js`

To create this repo,

```
npm init -y
npm i typescript -D
```

To create build folder,

`npm run build`

To directly run the TS files,

```
npm i ts-node
npx ts-node src/<>.ts
```
